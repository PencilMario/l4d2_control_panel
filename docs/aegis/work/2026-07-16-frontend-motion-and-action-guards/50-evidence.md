## 兼容与漂移检查

- HTTP API、载荷、后台任务轮询、页面导航和确认步骤保持不变。
- 动效由 CSS 承担，未引入第三方动画运行时。
- 异步锁由功能组件或共享任务创建入口持有；不同实例和不同资源的操作保持独立。
- 读取、导航、弹窗开关和普通刷新没有纳入写操作锁。
- `prefers-reduced-motion: reduce` 会将非必要动画和过渡缩短到近乎静态。

## 残余风险

- Vite 仍报告主 chunk 体积超过 500 kB。该提示在本次改动前已存在，适合单独进行路由或图表模块拆包优化。
- 视觉截图使用空实例初始状态；包含实例的桌面与移动业务流程由 Playwright 主流程覆盖，但未保存为长期基准截图。
